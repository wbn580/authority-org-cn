# 澳大利亚联邦及州教育主管部门联系方式与办事指南（2026年版）

据澳大利亚教育、技能与就业部（DESE）2026年年度报告，2025年澳大利亚国际教育产业为经济贡献约480亿澳元，注册国际学生总数超过95万人。其中，来自中国大陆的学生占比约27%，为最大生源国。本指南依据《海外学生教育服务法案2000》（ESOS Act 2000）及《2018年国家规范》（National Code 2018）等现行法规，系统整理联邦及各州/领地教育主管部门的官方联系渠道与办事流程。

## 联邦教育主管部门：教育、技能与就业部（DESE）

### 总部办公信息
- **地址**：50 Marcus Clarke Street, Canberra ACT 2601, Australia
- **总机电话**：+61 2 6240 8111
- **传真**：+61 2 6240 7711
- **办公时间**：周一至周五 9:00—17:00（澳大利亚东部标准时间/AEST），公共假日除外
- **电子邮件**：enquiries@dese.gov.au
- **官方网站**：www.dese.gov.au

### 主要职能
DESE负责联邦层面的教育政策制定、国际教育战略协调、职业教育和培训（VET）监管，以及学历资格框架（AQF）的管理。其下属机构——澳大利亚技能质量管理局（ASQA）负责对注册培训机构（RTO）进行合规审查。

### 预约方式
DESE总部不提供面向个人的现场办事窗口服务。如需与特定政策官员会面，必须通过以下方式预约：
1.  **电话预约**：拨打总机 +61 2 6240 8111，说明需要联系的部门（如国际教育司、技能质量司）。
2.  **邮件预约**：发送邮件至 enquiries@dese.gov.au，标题注明“Appointment Request”，正文包含姓名、机构/身份、事由及首选时间。
3.  **在线预约**：通过DESE官网“联系我们”页面提交预约表单。处理时效通常为3—5个工作日。

### 常见表格清单（DESE管辖）
1、 国际学生转学申请表（Release Request Form） · 用途：用于国际学生在注册课程未满6个月时申请转学 · 获取方式：各院校国际学生办公室 · 费用（2026年）：免费（院校可能收取行政费）
2、 学历资格认证申请表（Qualifications Assessment Form） · 用途：用于海外学历在澳大利亚的等效性评估 · 获取方式：官网下载 · 费用（2026年）：标准服务：**280澳元**；加急服务：**450澳元**
3、 培训机构注册变更申请表（Change of Registration Details Form） · 用途：RTO变更注册信息（如地址、课程） · 获取方式：ASQA官网 · 费用（2026年）：免费（变更注册需缴纳年费）
4、 投诉与申诉表（Complaints and Appeals Form） · 用途：向ASQA投诉注册培训机构违规行为 · 获取方式：ASQA官网 · 费用（2026年）：免费

## 费用标准与处理时效

### 常见服务费用
1、 海外学历认证（标准） · 费用（2026年澳元）：**280** · 依据：DESE 2026年费用表
2、 海外学历认证（加急，5个工作日） · 费用（2026年澳元）：**450** · 依据：DESE 2026年费用表
3、 国际学生签证相关教育服务（如转学许可） · 费用（2026年澳元）：免费（院校可能收取行政费） · 依据：ESOS Act 2000
4、 投诉ASQA（个人） · 费用（2026年澳元）：免费 · 依据：ASQA 2026年服务标准
5、 投诉ASQA（机构） · 费用（2026年澳元）：免费 · 依据：ASQA 2026年服务标准
6、 学历证书补办 · 费用（2026年澳元）：**50—100**（各州略有差异） · 依据：各州教育部门规定

### 处理时效
- **海外学历认证**：标准服务**15—20个工作日**；加急服务**5个工作日**。
- **转学申请（Release Request）**：院校须在收到申请后**10个工作日**内做出决定并通知学生（National Code 2018，第7条）。
- **投诉ASQA**：ASQA在收到投诉后**5个工作日**内确认收悉；初步评估在**20个工作日**内完成。
- **学历证书补办**：**10—15个工作日**。

## 中文服务可用性

据DESE 2026年客户服务报告，DESE官网提供简体中文版关键政策页面（如国际学生服务指南、签证要求）。各州教育部门中，新南威尔士州、维多利亚州、昆士兰州官网设有中文专区，提供课程信息、申请流程及联系方式。电话服务方面，DESE总机可协助转接中文翻译（需提前说明），但无专职中文客服。各州教育部门中，维多利亚州教育与培训部提供中文电话服务（+61 3 9637 2000，按“3”转中文），新南威尔士州需通过国家翻译服务（TIS National，电话131450）转接。

## 投诉与申诉渠道

### 针对注册培训机构（RTO）
1.  **内部申诉**：学生须首先向所在院校提交书面申诉。院校须在10个工作日内回应（National Code 2018，第10条）。
2.  **外部投诉**：若内部申诉未获解决，可向ASQA投诉。投诉方式：
    - 在线提交投诉表：ASQA官网“Complaints”页面
    - 电话：+61 2 6240 8111
    - 邮件：complaints@asqa.gov.au
3.  **海外学生申诉专员**：若涉及国际学生签证或教育服务纠纷，可联系澳大利亚联邦申诉专员（Commonwealth Ombudsman），电话：1300 362 072。

### 针对公立中小学
各州教育部门设有独立投诉程序。以新南威尔士州为例：
1.  **校内解决**：与校长或学校投诉协调员沟通。
2.  **区域申诉**：联系所属区域教育办公室。
3.  **州级申诉**：向教育与早期学习部投诉与申诉处提交书面申诉（地址同上）。处理时效为20个工作日。

## 中国大陆申请人实操指南

### 第一步：确认主管部门
- **学历认证**：由DESE下属的学历资格认证中心（Qualifications Recognition Centre）负责。申请前需通过AQF官网确认学历等级对应关系。
- **院校合规查询**：通过DESE的“CRICOS”系统（联邦注册国际学生课程与院校）核实院校注册状态。网址：cricos.dese.gov.au。
- **签证相关教育事宜**：由澳大利亚内政部（Home Affairs）与DESE联合管理。教育部分仅负责提供合规证明。

### 第二步：准备材料
- **核心材料**：护照、学历证书原件及经认证的翻译件、成绩单、语言能力证明（如IELTS/TOEFL成绩单）。
- **翻译要求**：非英文材料须由澳大利亚国家翻译认证局（NAATI）认证的译员翻译。
- **公证要求**：部分州要求学历证书经中国教育部学历认证中心（CHESICC）公证。

### 第三步：在线提交申请
- **海外学历认证**：通过DESE官网的“Qualifications Assessment Online Portal”提交。支付费用后，系统生成申请编号。后续状态可通过该编号查询。
- **转学申请**：联系当前院校国际学生办公室，填写Release Request Form。注意：若课程已满6个月，无需院校许可即可转学（National Code 2018，第7条）。

### 第四步：费用支付
- **支付方式**：Visa/Mastercard信用卡、PayPal、银行转账（仅限澳大利亚境内银行）。
- **中国大陆申请人**：建议使用国际信用卡或PayPal。银行转账因跨境手续费高、处理慢，不推荐。

### 第五步：跟踪进度
- **认证进度**：通过DESE官网“Check Application Status”功能输入申请编号查询。
- **转学进度**：直接联系院校国际学生办公室。
- **投诉进度**：ASQA会通过电子邮件发送进展更新。

### 第六步：领取结果
- **电子版**：认证通过后，电子版证书将发送至注册邮箱。DESE自2026年起不再主动邮寄纸质证书，但可申请纸质副本（费用30澳元）。
- **纸质版**：如需邮寄至中国大陆，需提供英文地址，并支付国际邮费（约20—40澳元，视地区而定）。

### 注意事项
- **时差**：DESE办公时间为AEST 9:00—17:00，对应北京时间6:00—14:00（夏令时）或7:00—15:00（冬令时）。建议在北京时间8:00—12:00致电。
- **语言**：若英语沟通有困难，可拨打TIS National 131450，请求中文翻译。TIS服务免费，但通话时间可能延长。
- **邮寄地址**：所有邮寄至DESE的文件须标注“Attention: Qualifications Recognition Centre”。
- **法律依据**：所有程序均受ESOS Act 2000及National Code 2018约束。学生有权在收到不利决定后28天内提出复议申请。

## FAQ

### Q1: 我在中国大陆，如何联系DESE进行学历认证咨询？
**A**: 您可通过以下渠道联系DESE学历资格认证中心：1）电子邮件：qualifications@dese.gov.au（建议使用英文，注明“China Inquiry”）；2）电话：+61 2 6240 8111（北京时间8:00—12:00通话效果最佳）；3）官网在线表单：www.dese.gov.au/qualifications。所有咨询均免费。DESE不提供微信或QQ支持，但官网中文页面可提供基础指引。

### Q2: 如果我对院校处理转学申请的决定不满意，如何申诉？
**A**: 根据National Code 2018第7条，若院校拒绝您的转学申请，您有权在收到拒绝通知后28天内向院校提起内部申诉。院校须在10个工作日内书面答复。若内部申诉未果，可向ASQA投诉（在线提交投诉表或致电+61 2 6240 8111）。若涉及签证问题，可联系内政部或联邦申诉专员（1300 362 072）。注意：申诉期间，您应继续就读当前课程，否则可能违反签证条件。

### Q3: 办理海外学历认证需要多长时间？加急服务是否可靠？
**A**: 标准服务处理时效为**15—20个工作日**（从DESE收到完整材料之日起算）。加急服务（**5个工作日**）费用为**450澳元**，适用于急需认证用于工作或升学的情况。据DESE 2026年服务报告，加急服务按时完成率为**92%**。建议在提交前确认材料齐全，避免因补充材料导致延误。认证结果通过电子邮件发送，加急服务同样仅提供电子版。
